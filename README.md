MeetIQ is an AI-powered meeting analyzer that automatically transcribes audio/video recordings in any language (including Tamil, Hindi) and generates a complete English summary with speaker detection, action items, decisions, sentiment analysis, and risk identification.
It features a full-stack web app with real-time live analysis, auto email summaries to participants, and a CLI tool — all processing done locally with no paid transcription service.
Tools used: Python, FastAPI, OpenAI Whisper, Groq API (LLaMA 3.3 70B), NumPy, FFmpeg, HTML/CSS/JavaScript, Gmail SMTP.
